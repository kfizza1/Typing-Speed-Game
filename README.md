# Typing Speed Game

A simple typing speed test built with HTML, CSS, and JavaScript.

## Features
- Starts timer on first keystroke  
- Calculates Words Per Minute (WPM)  
- Shows time taken  
- Reset button to try again  

## Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## How to Run
1. Open the HTML file in a browser
2. Start typing the actual sentence in lowercase
3. Complete it to see your typing speed

## Purpose
Practice JavaScript logic, DOM manipulation, and event handling.
